linux: time zone

all the info is available at https://wiki.archlinux.org/title/System_time. below are the commands i used in my personal case.

show current time zone for the system:

timedatectl status

list availabe time zones w/:

timedatectl list-timezones

set time zone w/:

timedatectl set-timezone <Zone/SubZone>
